# Mitukula Nikhitha - Personal Portfolio

A sleek, responsive, and highly interactive developer portfolio showcasing over 8+ years of experience in .NET full-stack and mobile app development. Live hosted on GitHub Pages.

🌐 **Live Site:** [https://nikhithamitukula.github.io](https://nikhithamitukula.github.io)

---

## 🚀 Key Features

* **Premium Bento Grid Design:** Modern dark mode interface styled with warm gold accents.
* **Typing Animation:** Dynamic and smooth intro header using `Typed.js`.
* **Curved Floating Navbar:** Top navigation bar that shrinks into a centered, glassmorphic pill on scroll.
* **On-Page Form Validation:** Native validation of input fields showing clean error/success alerts in the Contact form.
* **Interactive Tooltips & Copy-to-Clipboard:** Quick links to instantly copy email or phone details.
* **One-Click Resume Download:** Easy PDF download access in the hero section and navigation bar.
* **Responsive Layout:** Tailored with fluid media queries to look premium on desktops, tablets, and mobile devices.

---

## 🛠️ Tech Stack & Technologies

* **Core:** HTML5, CSS3, Vanilla JavaScript
* **Dynamic Animations:** Typed.js
* **Icons:** FontAwesome v6
* **Typography:** Outfit (Google Fonts)
